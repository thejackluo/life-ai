## Technical Details (If Asked)

**Tech Stack:**
- Python 3.11+
- OpenAI API (GPT-4o)
- Pydantic for data models
- JSON file storage
- Terminal UI

**Message Processing:**
1. Export messages to JSON
2. LLM analyzes full conversation history  
3. Extracts personality, speaking style, interests
4. Generates structured Character profile
5. At runtime, LLM uses profile + message history for responses

**Architecture Highlights:**
- Modular Python structure
- Clean separation: models, game logic, LLM integration
- File-based state management (simple, works)
- 128k token context window (fits full message histories)
